# Movie Recommendation System
Capstone Project Coding Camp 2026 powered by DBS Foundation (Tim CC26-PSU255)

## Tentang Proyek
Proyek ini adalah sistem rekomendasi film berbasis sinopsis menggunakan 
Sentence-BERT dan TensorFlow. Selain model AI, proyek ini juga punya 
dashboard interaktif untuk eksplorasi data film.

## Link Dashboard
https://movie-recommendation-v3mpy94box2ytktuuqmhbj.streamlit.app/

## Dataset yang Digunakan
- TMDB 5000 Movie Dataset (Kaggle)

## Library yang Digunakan
pandas, matplotlib, seaborn, streamlit, scikit-learn, tensorflow, sentence-transformers
